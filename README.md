# Linked-Data-Project-2026
Linked data project for LS563: Linked Data. This project contains a dataset focusing on early episodes of the Scooby-Doo series.

This linked data project contains a total of 89 episodes from the early installments of the Scooby-Doo series. This project, while containing important information regarding series, season, episode title, rating, network, and more, is also focused on linking the many different monsters and culprits that appear throughout Scooby-Doo. 

To express this dataset, the Schema.org controlled vocabulary and Schema.org ontology for episode were used. Schema.org's controlled vocabulary helped define labels used within the project while Schema.org's ontology helped structure the way properties connected to Scooby-Doo episodes. The project's linking strategy focused on expressing values that did not require linking, such as date and duration, as literals while values such as series, season, and network were given URIs to help differentiate them from other series, seasons, and networks related to Scooby-Doo. URIs were also given, when available, to the monsters and culprits within each episode to better connect them to the episodes they belong to. 

Wikidata contains the most information regarding Scooby-Doo, so Wikidata is used for the URIs of most properties. Network, unlike the other properties, uses VIAF for its URI. Examples of triples that appear in the project include What a Night for a Knight - schema:contributor - CBS, What a Night for a Knight - schema:partOfSeason - Scooby-Doo Where Are You! season 1, and Black Knight Ghost - schema:sameAs - Mr. Wickles. 

This project's goal is to help users be able to conduct research on the Scooby-Doo series.
